# 入门教程

## 安装
使用命令行工具安装最新稳定版：

	sudo npm -g install sails

在 `Windows` 上(或者在有`Homebrew`的`Mac OS`系统上)，不需要使用`sudo`:

	npm -g install sails

## 创建`sails`新工程
创新新应用:

	sails new testProject

启动服务器:

	cd testProject
	sails lift

这时，如果你访问(http://localhost:1337/) 你会看到默认主页。
